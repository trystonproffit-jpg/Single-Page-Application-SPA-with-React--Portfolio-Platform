# Project Showcase SPA

A simple single-page application (SPA) built with **React** and **Material UI**. Users can add, view, search, and delete projects with titles, descriptions, and GitHub links.

## Features

- Add new projects with a title, description, and GitHub link
- View a list of projects
- Search projects by title
- Delete projects
- Responsive design with Material UI cards

## Getting Started

### 1. Clone the repository
```bash
git clone git@github.com:trystonproffit-jpg/Single-Page-Application-SPA-with-React--Portfolio-Platform.git 
```
### Install dependencies
```bash
npm install
```

### 3. Start the Dev server
```bash
npm run dev
```

### 4. Open in your browser
